# Git 常用指令
## 1. 開新 branch	

	$ git branch 新分支名 

## 2. 切換 branch
	$ git checkout 分支名
	
## 3. 新開 branch 並切換到新 branch
	$ git checkout -b 新分支
	
## 4. 查看 branches
	$ git branch -a
	
## 5. 查看遠端 branches
